# Bank Management System

A desktop banking management system developed using C# (.NET Framework 4.8) and SQL Server.

The system manages clients, banking transactions, users, permissions, and currency exchange operations through a modern WinForms interface built with Guna UI.

### Architecture

- Presentation Layer (WinForms)
- Business Logic Layer (BLL)
- Data Access Layer (DAL)

## Features

### Authentication

- Secure login system
- Show / Hide password
- Remember Me support
- Login activity tracking
- Real-time clock

### Client Management

- Add, update, delete, and search clients
- Account number lookup
- Input validation

### Transactions

- Deposits and withdrawals
- Client-to-client transfers
- Transfer logging
- Total balances reporting

### User Management

- User administration
- Login register
- Role-based permissions
- Bitwise permission system

### Currency Exchange

- Currency management
- Exchange rate updates
- Currency calculator
- Search by country name or code

## Technologies

- C#
- .NET Framework 4.8
- SQL Server
- WinForms
- Guna.UI2.WinForms
- ADO.NET

## Concepts Practiced

- 3-Tier Architecture
- RBAC (Role-Based Access Control)
- Bitwise Permissions
- Database Design
- Transaction Processing
- Audit Logging
- Data Validation

## Purpose

This project was built to practice developing a complete banking system that combines database design, business logic, user management, financial transactions, and desktop application development.
